# Simple-blog-page

This project is a static blog page displaying 2–3 sample posts. It is designed using HTML and CSS with improved styling and readability.

# Features
3 sample blog posts with title, author, and date
Clean and readable layout
Google Fonts integration (Poppins)
Hover effects for better UI interaction
Background color styling for modern look
# Technologies Used
HTML5
CSS3
Google Fonts
# Learning Outcomes

Through this project, I learned how to structure blog content, enhance typography using Google Fonts, and improve user experience using hover effects and clean UI design.
